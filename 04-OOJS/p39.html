JAvascript Encapsulation

It is process of building the data with the function acting on that data.It allows us to control the data Validate it.

Points to remember..!
  • Use var keyword to make data member private.
  • USe setter method to set the data and get method to get the data.

Encapsulation allow us to handle an object using the following properties.

Read/Write- Here ,we use setter method to write the data and getter methods read that data.

Read Only-In this case ,we use getter methods only

Write Only-In this case ,we use setter methods only
